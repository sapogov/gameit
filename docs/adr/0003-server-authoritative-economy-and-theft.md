# Server-Authoritative Economy and Theft

Status: accepted

Farm production, collection, Theft, Theft Cooldowns, Guard Battles, guard Fainted state, and Theft Logs are server-authoritative. Clients may send intent, but the server validates ownership, visitor permissions, stored resources, guard state, cooldowns, battle outcomes, stolen amounts, and owner log entries. This protects the resource economy and PvP theft loop from client-side manipulation.
