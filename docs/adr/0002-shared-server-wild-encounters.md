# Shared Server Wild Encounters

Status: accepted

Wild Encounters are visible shared server entities, not client-personal random encounters. The first valid server interaction claims the encounter, defeated entities respawn after 90-120 seconds by rerolling from the zone table, and losing or successfully running away applies a 30-second per-player cooldown for that entity. This preserves multiplayer consistency, prevents duplicate farming, and keeps the world legible while leaving low-chance movement encounters for later.
